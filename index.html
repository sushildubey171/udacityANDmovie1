Welcome to the popular movie android app world.

This project is the part of Udacity Android Nanodegree curriculum. It's an Android app which display the popular and famous movie trending now-a days. It shows movie poster, release date, user rating and reviews. It only for learning purpose. Before running this application user needs to get one api key from themoviedb.org and enter at appropriate place. It's only the initial version of this app.
